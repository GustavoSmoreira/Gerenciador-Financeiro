## 🎯 Sobre o Projeto

O **Gerenciador Financeiro** é uma aplicação voltada para o controle de finanças pessoais ou empresariais. O objetivo principal do projeto foi estruturar um domínio limpo e aplicar boas práticas de desenvolvimento de software, garantindo um código escalável e de fácil manutenção.

### ✨ Principais Funcionalidades
* **Cadastro de Transações:** Entrada e saída de valores categorizados.
* **Listagem Dinâmica:** Visualização clara do histórico de movimentações.
* **Cálculo de Saldo Automático:** Processamento em tempo real do balanço financeiro com base nas receitas e despesas.

---

## 🛠️ Conceitos Técnicos Aplicados

Para os recrutadores técnicos, aqui estão os pilares de arquitetura e linguagem que utilizei neste projeto:

* **Encapsulamento e Abstração:** Proteção dos dados sensíveis das transações e exposição apenas de métodos necessários.
* **Collections Framework (`List`, `Map` ou `Set`):** Utilizado para o gerenciamento de dados em memória de forma eficiente.
* **Separação de Responsabilidades:** Divisão clara entre as entidades de domínio (`Entities`), lógica de negócio (`Services`) e a execução do sistema (`Application`).
